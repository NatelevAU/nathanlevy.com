## Setup

```bash
sudo apt update
sudo apt install curl
sudo curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt install nodejs build-essential libssl-dev
npm install
```

## Running the app

```bash
npm run dev
```
